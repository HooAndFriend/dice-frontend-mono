# Dice Project Mono Repo

Dice Project의 Frontend Monorepository 입니다.

### Version

```
Next : 13
Node : 18.18.2
pnpm : 8.11.0
```

### Using Stack

```
Next13, storybook, pnpm, turborepo, tailwind, rtk, rtk query
```

### 필독

```
해당 프로젝트는 pnpm을 이용합니다. 사전에 pnpm을 설치해주세요.

참고 : https://pnpm.io/ko/installation
```

### Run

```
1. git clone

2. pnpm install

3. pnpm dev
```
