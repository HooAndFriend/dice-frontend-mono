# React Boilerplate

### Stack Version

```
React : 18
node : 16.18.0
React-router-dom V6
```

### 실행 방법

```
1. git clone
2. cd dice-frontend
3. yarn
4. yarn dev
```

### Project Using

```
React, React-router-dom, Redux, Redux-Tool-Kit, Redux-Tool-Kit-Query, MUI, Typescript
```

### Build Using

```
Vite, Vercel, SWC, Docker
```
